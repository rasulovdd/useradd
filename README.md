<h1 align="center">useradd</h1>

## Скрипт управления пользователями Linux

Этот скрипт автоматизирует создание и удаление пользователей в Linux‑системе с расширенной настройкой окружения.

## Функциональность
Скрипт позволяет:

- Создать нового пользователя с домашней директорией и оболочкой /bin/bash.

- Установить пароль (с проверкой совпадения).

- Настроить SSH‑доступ (добавление публичного ключа).

- Предоставить права sudo (по запросу).

- Настроить удобное окружение:

    - Алиасы (ll, gs, цветные grep).

    - Красочный prompt (PS1).

    - Функцию extract для распаковки архивов.

    - Приветственное сообщение при входе.

- Удалить пользователя с опцией сохранения/удаления домашней папки.

## Требования
- ОС: Linux (совместимо с Debian/Ubuntu, CentOS/RHEL и аналогами).

- Права: запуск от имени root или через sudo.

## Установка

1. Скачайте репозиторий<br/>

    ```bash
    git clone https://github.com/rasulovdd/useradd.git && cd useradd
    ```

2. Даем права на запуск 
    ```bash
    chmod +x create_user.sh
    ```
    
3. запускаем)

    ```bash
    ./create_user.sh
    ```

4. либо запускаем одной командой (нужны права sudo)

    ```bash
    bash <(curl -fsSL https://raw.githubusercontent.com/rasulovdd/useradd/main/create_user.sh)
    ```

## Настройка окружения

При создании пользователя в ~/.bashrc добавляются:

1. Алиасы:

    - ll — подробный список файлов (ls -alF).

    - gs — статус Git (git status).

    - Цветные grep, egrep, fgrep.

2. Красочный prompt (PS1):

    - Разные цвета для хоста, пользователя, текущей директории.

3. Функция extract

    Позволяет распаковывать архивы одной командой:

    ```bash
    extract file.tar.gz file.zip
    ```
    
    Поддерживаемые форматы: .tar.bz2, .tar.gz, .bz2, .rar, .gz, .tar, .zip, .7z.

4. Приветственное сообщение
    
    При входе выводится:

    -  Имя пользователя.

    - Текущая директория.

    - Дата и время.

5. Безопасная работа 

    чтобы ваша команда выполнялась даже если связь прервется используйте команду 
    ```bash
    run
    ```
    перед использованием установите tmux
    ```bash
    sudo apt install tmux
    ```
